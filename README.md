# To_Do_List

## Overview

**● This project includes the ability to add, delete, and view tasks, as well as save and load task lists from a file.**  
**● It uses various Python concepts such as object-oriented programming (OOP), reading and writing CSV files, and data management.**  
### Features  
**Add new task: The new task is added to the list.**  
**Delete task: The task is removed from the list.**  
**View task list: All tasks in the list are displayed.**  
**Save task list to CSV file: All tasks are automatically saved to a CSV file.**  
**Load from CSV file: The task list is automatically loaded from a CSV file.**  
**Choose a priority for tasks: Each task can have a priority (for example, high, medium, or low).**
